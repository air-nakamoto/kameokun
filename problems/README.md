# problems ディレクトリ

亀夫君問題の本文、答え、解説、レビュー用メモを Markdown で管理します。

## 追加方法

1. `template.md` をコピーする
2. ファイル名を `YYYY-MM-DD-short-title.md` に変更する
3. メタ情報、問題文、答え、解説、レビュー観点を埋める
4. Pull Request を作ってレビューを受ける

## レビュー観点

- 問題文だけで必要な条件が読み取れるか
- 答えが複数に分かれないか
- 亀夫君らしい雰囲気や文体になっているか
- 解説が短くても理解できるか
- 公開前に確認すべき注意点が残っていないか
