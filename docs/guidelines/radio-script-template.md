## ⑧進行

**※自由にゲーム進行**
・悩める相談者に気になる点を色々質問
・キャラが知っていることは答える。（言いづらいことを濁したり、知らないことは答えない）
・「出題者呼び出し」はどんなタイミングでも自由に可能
**※初回の「出題者呼び出し」発生時には、ルール説明。**

🐢　**出題者呼び出しでは、**
「今までの会話の中で、重要な部分があったかどうか？」を確認することができます。
　キャラクターには直接聞けないことも、自由に相談できます。
　ただし、「解決策そのもの」はお伝えできません。
　情報の振り返りや、作戦会議の場として、お使いください。

（重要な会話確認・進行相談）

🐧**【回答者側：出題者呼び出し】
※基本的に自由に相談**
・今までの進行で思ったところ（印象、感じたことなど）自分の思考を開示をする。
・わかった情報まとめ
・重要な会話があったか確認
・進行に関して相談
・今後の方針を宣言してみる
・会話したい人の呼び出し依頼

**🐢【出題者側：出題者呼び出し】**
**※フェアに回答（新事実をなるべく出さない）
　でも、テンポよく進行できるようにヒントは出していく**
・キャラには聞けない「重要」「重要じゃない」は、開示する
・キャラに聞かないとわからないことは、なるべく情報開示を避ける
・相談→進行の相談に乗る
　　　　真相に確信に迫れるようなヒント
　　　　ぐだ付き防止のため、周り道をしていて部分は、やんわり避けるようヒント
