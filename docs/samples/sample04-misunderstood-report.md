# 亀夫君問題サンプル4「伝わらない報告」

**※亀夫君問題※ 出題時の[事前準備詳細リスト]**


## 問題概要
**タイトル**: 「伝わらない報告」
**構造**: 2重謎（報告書問題→先輩の付箋行動の謎）
**ジャンル**: ビジネスコミュニケーション＋ヒューマンドラマ
**想定時間**: 30-45分

## 問題文
こんにちは...実は今、すごく困っていることがあって...。

私、健太と申します。最近会社でのコミュニケーションはチャットやメールでがほとんどのため、どうも先輩社員とのコミュニケーションがうまくいっていません。

特に、チャットでの業務報告が「要点がわからない」「結局どうしてほしいの？」と何度も指摘されてしまい、すっかり自信をなくしてしまいました。

「自分ではちゃんと書いているつもりなのに、どうして伝わらないんだろう...」
「どうすれば、先輩に分かりやすく、的確な報告ができるようになるんだろうか...」

本当に困っているんです...どうかアドバイスをお願いします。

## 登場人物設定
**健太（けんた）**: IT企業2年目、真面目で向上心あり
- 報告書：時系列で詳細に書くのが良いと思い込み
- 先輩との関係：尊敬しているが「そっけなくて怖い」と感じている
- 付箋発見：先月の机整理時から（先輩の不器用な優しさに気づいていない）

## 2重謎構造

### 第1の謎：報告書問題
**表面的問題**: 報告書が「要点がわからない」と言われる
**真相**: 健太は時系列重視、先輩は「結論＋依頼内容」を求めている
**解決策**: 「結論→依頼→経緯」の順番に変更
**学び**: 相手の立場で考える、行動順と報告順は逆

### 第2の謎：先輩の付箋行動
**表面的問題**: 机の隠れた場所に謎の付箋（「了解」＋下手なイラスト）
**真相**: 先輩が後輩指導で悩み、上司のアドバイスで付箋コミュニケーションを実践
**解決策**: お互いの思いやりを理解し、健太から歩み寄る
**学び**: 不器用でも相手を思う気持ちの大切さ

## コピペ用素材

### 悪いメール例
```
【件名】A社様のログイン不具合の件について

〇〇先輩
お疲れ様です。健太です。

本日10:00頃、A社の佐藤様より、管理画面にログインできないとのご連絡をメールでいただきました。

まず、ご連絡いただいた内容を元に、当社のテスト環境で同様の現象が再現されるかを確認しました。
ユーザーID「testuser」と指定されたパスワードで試したところ、私の環境では問題なくログインできました。

次に、本番環境のサーバーログを確認しました。
10:00前後のA社様のIPアドレスからのアクセスログを調べたところ、確かにログイン試行の記録があり、認証エラーとなっていました。

エラーコードは「E-5003」でした。
このエラーコードについて社内ドキュメントを検索したところ、「パスワードの連続間違いによる一時的なアカウントロック」の可能性があることが分かりました。

以上、ご確認のほどよろしくお願いいたします。
```

### 改善されたメール例
```
【件名】A社様のログイン不具合（アカウントロックの可能性・ご確認のお願い）

〇〇先輩
お疲れ様です。健太です。

本日ご連絡があったA社様のログイン不具合の件、調査したところパスワードの連続間違いによるアカウントロックの可能性が高いです。

つきましては、お忙しいところ恐縮ですが、以下2点についてご対応・ご指示をいただけますでしょうか。

【お願いしたいこと】
1. A社様のアカウント状況のご確認と、ロックされていた場合の解除
2. お客様へのご案内方法についてのご指示
   （例：「システム側でロックを解除しましたので、再度お試しください」とお伝えして良いか、など）

【調査経緯】
・本日10:00頃、A社様よりログイン不可との連絡あり。
・サーバーログを確認したところ、認証エラー（E-5003）の記録がありました。
・社内ドキュメントによると、同エラーはアカウントロックを示唆するものでした。

お忙しいところ恐縮ですが、ご確認のほどよろしくお願いいたします。
```

## シナリオ分岐例

### チャット・メール系
**「感謝のチャットを送ったら？」**
→ 先輩から「了解」とだけ返信、でも翌日の付箋にいつもより大きめのイラスト

**「『付箋、気づいてます』と伝えたら？」**
→ 先輩から「？？？何のことですか」と返信、その日から一時的に付箋が止まる

### 他の人に聞く系
**「同僚に先輩のことを聞いたら？」**
→ 「真面目だけど不器用、本当は後輩思い」という情報

**「上司に相談したら？」**
→ 「先輩が後輩への接し方を気にして相談に来ている」ことが判明

### 避けるべき行動
**「付箋をやめてほしいと言ったら？」**
→ 先輩がショックを受けて完全に付箋停止、チャットも最小限に

## 重要な移行フレーズ
「書き方については、本当にスッキリしました！...ただ、実はもう一つだけ、どうしても気になっていることがあるんです。これは全然仕事の相談じゃないんですけど...その先輩のことで、すごく不思議なことがあって...」

## 解決の鍵
- 第1の謎：「相手が何を知りたいか」を考える視点転換
- 第2の謎：「不可思議な行動」→「愛おしい行動」への感情転換
- 最終的に：健太が「支えられる側」から「支える側」へ成長

## 成功指標
- 実用的なビジネススキル向上
- 人間関係への理解深化
- 感動的な気づきの瞬間
- 相互理解とお互いの成長
