# bot ディレクトリ

Claude Project の「手順」欄や、ボットの振る舞いを決めるガイドラインを管理します。

## 現在の構成

- `kameo-guidelines.md`: 新作オリジナル亀夫君問題出題専用ボットの統合ガイドライン
- `kameo-templates.md`: 出題開始、出題者呼び出し、解決判定、感想戦などの会話テンプレート
- `nakamoto-style-guide.md`: 中本アイアールの人格、口調、価値観、表現ルールのガイド

## 運用方針

- プレイ中に見せてはいけないネタバレ情報と、解説モードで開示してよい情報を分けて管理します。
- 新作問題作成モードと既存問題出題モードの違いを明確にします。
- Claude Project から貼り付けられた重複データは、完全重複を統合してから保存します。
